Work items (v2)
WI-1 · Migrazione schema
	•	Tabelle: quotes (con access_token, sent_email_log, client_response_log come jsonb), quote_items, quote_supplier_markups.
	•	Colonne profiles: brand_logo_url, brand_primary_color, brand_secondary_color, default_markup_percent.
	•	Enum quote_status: BOZZA, INVIATO, ACCETTATO, RIFIUTATO, CONVERTITO_IN_CONTRATTO.
	•	Stima: 3 ore.
WI-2 · Policy RLS
	•	Policy quotes SELECT/UPDATE/DELETE solo capostipite owner.
	•	Policy quote_items gestita tramite quotes (se vedi la quote, vedi i suoi items).
	•	Policy public su quotes: accesso tramite access_token non richiede auth ma richiede token match (policy speciale "public read via token").
	•	Stima: 3 ore.
WI-3 · Trigger calcoli
	•	Trigger on INSERT/UPDATE/DELETE quote_items → ricalcola totali Quote (total_cost, total_client, margin).
	•	Funzione SQL calcola_markup_effettivo(quote_id, supplier_id, item_markup).
	•	Stima: 4 ore.
WI-4 · Enforcement limite 10 preventivi
	•	Trigger BEFORE INSERT su quotes se owner tier=FREE: SELECT COUNT WHERE status IN (BOZZA,INVIATO,ACCETTATO). Se >= 10 → RAISE EXCEPTION.
	•	Stima: 2 ore.
WI-5 · Edge Function quote-generate-pdf
	•	Riceve quote_id + variant (NEUTRA/PREMIUM).
	•	Query quote completa con items, supplier info, profiles per brand.
	•	Render HTML template Handlebars.
	•	Puppeteer headless → PDF.
	•	Upload su bucket 'quote-pdfs'. Path /{quote_id}/v{revision}.pdf.
	•	Restituisce signed URL.
	•	Stima: 8 ore.
WI-6 · Edge Function quote-send
	•	Orchestrazione: chiama generate-pdf, genera access_token, UPDATE quote status, invia email via Resend, INSERT calendar_entry IN_TRATTATIVA.
	•	Transazione SQL atomica via RPC.
	•	Stima: 5 ore.
WI-7 · Edge Functions pubbliche
	•	quote-public-preview: GET /p/preview/:token → restituisce dati preventivo se valid.
	•	quote-public-accept: POST /p/accept/:token → UPDATE quote + crea calendar_entry OPZIONATA + email capostipite.
	•	quote-public-reject: POST /p/reject/:token → UPDATE quote + email capostipite.
	•	Rate limiting via Supabase (config auth rate limit su endpoint pubblici).
	•	Stima: 6 ore.
