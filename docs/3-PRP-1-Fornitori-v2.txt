Contesto tecnico Supabase
Stack:
	•	Supabase (Postgres + Auth + Storage + Realtime + Edge Functions).
	•	Supabase CLI per migrazioni.
	•	Client JS/TS @supabase/supabase-js nel frontend.
	•	React 18 + TypeScript + Vite.
	•	UI: Tailwind + shadcn/ui.
	•	Test: Vitest, Playwright.
Convenzioni di codice:
	•	SQL: snake_case. File migrazioni: YYYYMMDDHHMMSS_nome.sql.
	•	TypeScript: strict. Tipi DB autogenerati con `supabase gen types typescript`.
	•	Componenti React: PascalCase. File: kebab-case.

WI-2 · Policy RLS
	•	Abilita RLS su tutte le 5 nuove tabelle.
